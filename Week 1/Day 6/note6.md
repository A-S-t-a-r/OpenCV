1. 均值滤波
   ```
   cv2.blur(image, (5, 5))
   ```
   + (5, 5): 滤波核的大小，表示在水平和垂直方向上取平均值的范围
   + 优缺点：去除图像中的随机噪声，但可能会导致图像边缘变模糊
2. 高斯滤波
   ```
   cv2.GaussianBlur(image, (5, 5), 0)
   ```
   + 0: 高斯核的标准差，如果为0，则根据核大小自动计算
   + 优缺点：去除图像中的高斯噪声，并且保留图像边缘信息
3. 中值滤波
   ```
   cv2.medianBlur(image, 5)
   ```
   + 5: 滤波核的大小，必须是奇数
   + 优缺点：去除图像中的椒盐噪声，并且保留图像边缘信息
4. 双边滤波
   ```
   cv2.bilateralFilter(image, 9, 75, 75)
   ```
   + 9: 滤波核的大小
   + 75: 颜色空间的标准差，控制像素值相似度的权重
   + 75: 坐标空间的标准差，控制空间距离的权重
   + 优缺点：除噪声的同时保留图像的边缘信息，常用于图像美化或预处理
