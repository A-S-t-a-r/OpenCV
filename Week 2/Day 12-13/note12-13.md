+ 目标追踪的基本流程为：
    1. 通过各种检测方法（颜色、形状、预训练模型等）获取目标位置
    2. 计算目标的质心或边界框
    3. 使用各种追踪算法进行连续帧的目标位置预测与更新
+ HSV颜色阈值分割
+ 连通域分析：connectedComponentsWithStats()（对于使用阈值分割后的二值图像可快速提取目标区域）
    ```
    int connectedComponents(InputArray image, OutputArray labels,int connectivity, int ltype, int ccltype);
    ```
    + image: 二值图像
    + labels: 输出图像
    + connectivity: 领域种类（4 or 默认8）
    + ltype: 输出标签类型（CV—_32s or 默认CV_16U）
    + ccltype: 标记连通域算法类型
+ 质心计算与追踪（例如使用图像矩、用轮廓的中心）
+ 目标跟踪：
  + 简单质心追踪（在连续帧中寻找最近质心）
  + 使用OpenCV内置追踪器（如KCF、MIL、CSRT等）
